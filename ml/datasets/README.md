# Datasets para Treinamento

Coloque os arquivos de dataset nesta pasta antes de executar o treinamento.

## CICIDS2017 (recomendado)

Dataset com tráfego de rede rotulado contendo ataques reais.

Download: https://www.unb.ca/cic/datasets/ids-2017.html

Arquivo esperado: `CICIDS2017_sample.csv`

## NSL-KDD (alternativo)

Download: https://www.unb.ca/cic/datasets/nsl.html

> Os arquivos CSV não são versionados no Git por serem grandes.
> Use Git LFS ou faça o download manual conforme os links acima.
